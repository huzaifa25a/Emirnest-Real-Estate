Emirnest is a full-stack property listing web application that allows users to explore and list properties for rent or sale. The application focuses on secure authentication, clean UI, and efficient property management.

---

## Features

- User registration and login with JWT authentication
- Protected routes for listing properties
- Create, view, and manage property listings
- Structured property details including price, location, and specifications
- Responsive UI built with modern frontend practices
- Contact form for user inquiries
- Static pages such as About Us and Terms & Conditions

---

## Tech Stack

### Frontend
- React
- Axios
- Tailwind CSS
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JSON Web Tokens (JWT)
